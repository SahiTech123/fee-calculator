# Kikokotoo cha Makato Tanzania

App ya kulinganisha makato, kuangalia usalama wa namba, na kuhesabu mikopo.

## Vipengele
- Kikokotoo cha makato (M-Pesa, Tigo, Airtel, AzamPesa, Selcom, HaloPesa)
- Kigunduzi cha namba za ulaghai
- Kikokotoo cha mkopo (4% kwa mwezi)
- Mwongozo wa kupata mkopo

## Jinsi ya Kuitumia
1. Fungua index.html kwenye browser
2. Ingiza kiasi na uone makato
3. Angalia usalama wa namba
4. Hesabu gharama ya mkopo

## Teknolojia
- HTML, CSS, JavaScript
- PWA (Progressive Web App)
